# Task Management App

A modern, responsive task management tool for small teams.

## Features
- **User Authentication**: Secure access (simulated).
- **Task Assignment**: Assign tasks to team members.
- **Status Updates**: Track progress (Pending, In Progress, Completed).
- **Deadline Reminders**: Visual cues for impending and overdue deadlines.
- **Premium UI**: Glassmorphism design with animated gradient background.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the app:
    ```bash
    npm run dev
    ```

3.  Open `http://localhost:5173` to view it in the browser.
